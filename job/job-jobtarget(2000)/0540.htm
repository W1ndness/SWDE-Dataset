In need of a Senior Test Engineer. The selected candidate will join a small team working to develop a test harness for the PLAYSTATION system.  The test harness is used to automate the testing of commercial security products. The selected candidate will be responsible for designing and developing test scripts using various scripting languages.  The selected candidate will work with other Test Engineers to create the test harness for the PLAYSTATION system. PLAYSTATION is a new effort that is currently in a prototype phase.   The system will leverage virtual machines to perform many of the tests executed by the test harness.  The overall design is still be architected, so the selected candidate will be joining the task at a phase where they can influence the overall design.

 *NOTE* Candidates last poly must be within the past 7 years.
