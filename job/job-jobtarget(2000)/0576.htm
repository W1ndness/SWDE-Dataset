Hyland Software is searching for a Database Programmer & Consultant to join their global Database Services department. This position is responsible for delivering database solutions, consulting and support pertaining to the OnBase® software and associated database technologies, in a manner that meets or exceeds customer expectations.  The position requires a solid understanding of the OnBase® database design, software query behavior, and OnBase® best practices related to database installation, administration, and maintenance.  A Database Engineer must also be committed to continued development of technical expertise in the aforementioned areas, as well as documenting and sharing this knowledge with coworkers.

Position Responsibilities:

  • Develop a thorough, detailed understanding of the OnBase® database schema and database operations associated with:
  • o Document objects and associated metadata
  • o Workflow actions
  • o Workview objects
  • o Import Processes
  • o Logged events
  • Analyze customer database implementations with respect to OnBase® solution performance
  • Assist HSI technical support personnel troubleshooting customer database issues
  • Make recommendations concerning the administration and maintenance of customer OnBase® databases
  • Execute training classes on OnBase® database administration and maintenance during Hyland Software conferences and other events
  • Document findings and recommendations resulting from providing database-related consulting services to customers and partners consistent with Hyland Software documentation standards
  • Consistently and proactively create new documentation for team knowledgebase containing relevant and actionable content to more efficiently troubleshoot database issues
  • Complete all mandatory certifications and training courses within the expected time frame
  • Provide database and document management conversion services in accordance with specific statements of work
  • Collaborate with Software Development and QA departments on database design and performance issues on a project level basis, as assigned.

Job Requirements
  • Bachelor's degree or equivalent experience
  • Clear, fundamental understanding of relational databases and industry-recognized concepts pertaining to SQL Server and/or Oracle database management systems
  • Professional oral and written communication skills, including the ability to read and write software documentation from a configuration or user interaction perspective
  • Ability to work independently and in collaboration with other business and technical professionals
  • Excellent Problem Solving Skills and Troubleshooting skills
  • Ability to work in a flexible, fast-paced environment
  • Provide exceptional follow through and be customer-service driven
  • Advanced interpersonal skills
  • Ability to work independently and in a team environment

Optional Requirements:

  • Demonstrated technical skills in database administration or design and one of the following additional areas: application development, networks, hardware configuration, or operating systems.
  • Experienced SQL Programmer (ANSI SQL, T-SQL, PL/SQL)
  • Performance Tuning Experience


Travel Expectations:

  • Commitment to travel approximately 50 percent of the time
