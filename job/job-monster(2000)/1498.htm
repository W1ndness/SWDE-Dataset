ETL Developer

About the Job

The extraction, transformation, and load (ETL) developer creates jobs moving data from system to system.  Source and target data often will be stored in enterprise-class databases,  such as SQL Server, but also include filed based items.  Jobs created by the ETL developer will undergo a QA and DBA evaluation prior to moving to production.   The ETL developer will be responsible for scheduling, monitoring, and responding to issues resulting from ETL job execution.  Corporate reporting and security standards must be followed.  The ETL developer will also be responsible for generating data quality metrics and overall summaries  of target data statistics, including their publication to metadata consumers.

ETL Developer will assist in the SSIS package creation, SQL Script creation, stored procedure development and report development, in support of decision support analysts. 

Technical Accountabilities:

*Ensure ETL code delivered is running, conforms to specifications and design guidelines.
*Unit testing ETL code to ensure it can be delivered and run in a system testing environment
*ETL job scheduling using SQL Server 2005 Agent
*SQL coding with SQL Server Management Studio
*Creation of T-SQL stored procedures
*Creation of reports suing SQL Server 2005 Reporting Services
*Facilitate database object code promotion using source control and communication with the DBA team
*Work with the business users to develop requirements for information delivery requirements


QUALIFICATIONS:

**5 yrs or more general ETL experience using tools such as Informatica, DataStage, SQL Server Integration Services.
**2 yrs or more ETL Development experience using SQL Server 2005 Integration Services
** Experience using .NET programming with SQL Server
**Experience working  in larger enterprise application integration environments
**Familiarity with database concepts such as normalization, indexing, physicala and logical modeling.
**Familiarty with the Kimball Methods of Data warehousing techniques and concepts.
**BS Computer Science or MIS required

Client requires US Citizenship or Green Card.  Local Candidates preferred- no relocation expenses.
