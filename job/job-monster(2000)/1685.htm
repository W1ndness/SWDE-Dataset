About the Job

HQ Systems Administrator

 

DESCRIPTION

 

Infrastructure Management:

·         Develop and maintain all system, application, security, and network configuration.

·         Maintain systems continuity of operations by developing and maintaining a back-up and disaster recovery plan.

·         Research, analyze, and implement upgrades, patches, new applications, equipment and emerging technologies.

·         Maintain and ensure compliance with corporate IT Security Plan.

·         Assist in short and long term planning including the development of strategic plans to leverage emerging technology to support future enterprise needs.

·         Assist in establishing and maintaining annual IT budget and solicits vendor quotes and recommends IT-related procurements.

 

Technical Operations:

·         Windows Administration – User management in Active Directory, DNS, and domain management.

·         Exchange Management – Exchange 2007 maintenance and configuration.

·         SharePoint Administration – Create web application and site collections in MOSS. Modify site settings, permissions, and themes.

·         Network Administration – Manage and maintain DHCP Server and be familiar with static and dynamic IP addressing. Monitors and troubleshoot network firewalls, switches, hubs, routers, and peripherals. DNS management of corporate owned domains.

·         Patch Management – Microsoft Windows Server Update Services (WSUS) management of Microsoft patches and updates.

·         Web Development - Design and maintain corporate website and associated intranets in compliance with Section 508 requirements.

·         Provide and document responsive and courteous tier 1 and 2 support and guidance to users.

·         Install and configure new IT-related hardware and software.

·         Sets up, configure, document, and support internal and/or external networks (LAN, VPN, Telecomm, & wireless).

·         Performs a variety of IT-related tasks to support corporate initiatives.

·         Troubleshoot printer/fax issues.

·         IT Service Management – Maintain ISO 20000 certification and pursue continual process improvement.

 

SKILLS

·         Server OS: Microsoft Server 2003, Linux – FreeBSD, Ubuntu Server

·         Workstation OS: Microsoft Windows XP, 7

·         Database Software: Microsoft SQL, MySQL

·         Web-Server Applications: Apache (FreeBSD), IIS (Window 2003)

·         Server Applications: Server 2003 Active Directory and DNS, Exchange 2007, SharePoint Portal 2003, Microsoft Office SharePoint, Deltek GCS Premier, Avaya IP Office

·         Programming Languages: HTML, CSS, PHP, ASP, JavaScript

·         Web Design Tools: Macromedia Dreamweaver/Flash, Adobe Photoshop/Illustrator

·         Business Applications: Microsoft Office 2007 Suite, Symantec (Anti-virus; Anti-Spam; Backupexec), Adobe Acrobat; SonicWall GUI, SharePoint Designer

·         Virtualization Software: VMWare ESXi, VMWare Server

·         Knowledge and Concepts: TCP/IP, HTTP/HTTPS, DHCP, NAT, SMTP, Open Source – Nagios, Cerberus, Client/Server Architectures, Apache Virtual Hosting, EVM interfaces, CMMI, ITIL/ISO 20K process improvement frameworks.

 

REQUIREMENTS

·         Bachelor's degree in Computer Science or related field.

·         4-6 years of relevant experience in a Windows Server environment.

·         CompTIA A+

·         MS certifications (MCSA, MCSE) preferred.

·         ITIL V3 Foundation, CISSP a plus
