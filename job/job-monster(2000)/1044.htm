About the Job

Johnston McLamb (JM) is an award-winning technology management services firm specializing in business intelligence, data visualization, and geospatial application development for government and private sector customers, including the U.S. Postal Service.
Headquartered in Chantilly, Virginia, the company recently joined CRGT Inc. to extend its services portfolio to include systems integration and operations for large-scale, high-volume solutions.

We continually seek Java Developers for full-time positions in Fairfax and Chantilly, VA



Job Summary: 

Design, develop, test and implement mission critical web applications using Java in an Oracle environment. Work with business owners and end users to analyze, define and document functional and technical business requirements to ultimately recommend appropriate design alternatives. Participate in defining application architecture and individual application and component designs, in addition to development tasks.


Job Requirements: 
  • Applications development in an Oracle 9i or 10g environment       
  • Developing and confirming data models      
  • UNIX, including shell scripts, CRON jobs and directory navigation      
  • CMMI level 2 project environment exp 
  • Application design, development and testing of database driven Java applications       
  • User interface / presentation layer development using JSP, Servlets, JavaScript, HTML, DHTML and CSS      
  • Full life cycle development of enterprise level applications       
  • Busiess Requirements Analysis
  • Development and deployment in a multi-tiered and multi-server application architecture 
  • Object oriented principles and concepts  
  • U.S. citizenship, Green card, or EAD status required AND cannot have traveled outside the U.S. for a cumulative total of 6 months over the last five (5) years.

 

 

Johnston McLamb offers competitive salaries and benefits package.  Please send your Word resume to: recruiting@johnstonmclamb.com  and indicate position code: JAVA
