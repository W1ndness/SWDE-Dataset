Senior Web Architect

About the Job

Unicity International is seeking a world-class Senior Web Architect with extensive web development experience and expertise in architecting, implementing, testing, optimizing and maintaining domestic and international web/mobile web sites and applications.

 

Platform/Language/Framework Expertise

·        Multiple web development programming languages (PHP, ASP, PERL, PYTHON, .NET, JAVA, RUBY, etc.)

·        Joomla or similar MVC Application Development Framework

·        JSON/REST and XML/SOAP based Web Services

·        Advanced knowledge of HTML, XHTML, CSS and XML

·        Implementing and consuming cross platform Web Services

·        Adobe Flex and Flash Technologies

·        One or more Mobile Application Frameworks, including PhoneGap, iWebkit, Sencha Touch, JQuery Mobile, and JQTouch

·        MySQL, SQL Server and Oracle

·        Amazon EC2 & S3

·        Windows and Linux environments

·        Apache and IIS configuration

 

 

Experience

 

·        Have personally designed, developed, tested and deployed multiple web applications, mobile web applications, and SEO-optimized websites

·        Have personally designed and developed reusable frameworks that accelerate solution development for web sites

·        Understand the mobile web application marketplace and how those mobile applications access and share data with back-end systems

·        Understand SOA concepts and have personally developed code that consumes web services (SOAP, XML Over HTTP, REST, XML-RPC)

·        Have personally developed cross-browser applications or components with one or more Javascript Frameworks, including Prototype, jQuery, YUI, ExtJS/Sencha, and MooTools

·        Have familiarity with source control systems (SVN, CVS)

·        Understand scalability and load-balancing issues with rich-media, Web 2.0 sites

 

 

Non-Technical Skills

·        Experience with cutting edge web collaboration design patterns and best practices

·        Experience in working in every phase of software development

·        Ability to evaluate, select and integrate technology from open-source, home-built and commercial sources to deliver a total, elegant solution

·        SCRUM Master Certification and/or multiple years’ experience working in an Agile development environment

Unicity offers a competitive salary, excellent benefits, and a great working environment. If you are interested in applying, please email resume with salary requirements to employment@unicity.net . Only those selected for an interview will be contacted.
