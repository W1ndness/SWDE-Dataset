Position Description

At MarketAxess, technology is our product.  Speed to market, commitment to excellence and creativity are vital to our success. Trade West Systems (TWS) in Salt Lake City is a subsidiary of MarketAxess.

TWS specializes in the area of ‘mid-office’ processing of security trading, providing access and connectivity to disparate external systems. As a senior developer, you will work together with technologists and client-facing teams. You will work on new initiatives, system enhancements, and bug fixes. You will help to design and plan new projects. You may also work on web projects.

The nature of our platform requires skill and experience in working on real-time, highly reliable, highly available transactional systems and relational databases. Our environment is entrepreneurial in nature, and we reward results, creativity, and initiative. A successful candidate must be highly motivated, intelligent and responsible.

Position Requirements

We require the following technical skills:

·         C# and .NET programming

·         Enterprise RDBMS

 

Experiences in the following technical skills are desirable but not required:

·         Network programming (TCP/IP, Message Queue, Web Services, other)

·         Microsoft SQL Server

·         Microsoft SQL Server Reporting Services

·         Microsoft .NET Foundation architecture (WPF, WCF, WWF)

·         Real time transaction processing

·         Multithreaded programming

·         Windows system administration and networking for Windows 2000 and later

·         Other programming languages (Java, C++, C)

·         User interface design (both rich client and web interfaces)

 

Other Requirements:

·      10+ years professional experience

·       Strong verbal and written communication skills as well as organizational and problem resolution
