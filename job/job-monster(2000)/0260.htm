Web Application Developer-SAP Systems

About the Job

      The web applications developer will design, develop and provide technical support for projects related to Arctic Cat’s Information Technology. This primarily will include Java development/support inside SAP’s Netweaver portal in support of Arctic Cat’s dealers and distributors and also .NET development/support in support of Arctic Cat’s suppliers.  This would also include building interfaces into the SAP systems either via SAP PI or via SAP RFC’s.  Candidate should have knowledge of Java, J2EE, C#, Javascript, HTML, XML and SOA.  SAP ABAP experience would be an added benefit.  The developer will work closely with the SAP business analysts/developers and other IT staff to maintain/build an integrated system.

  

Essential duties and responsibilities:  The person in the Web Applications Developer position will provide support to the functional areas of the company to which they are assigned by:

1.       Coordinating problem support and requests for enhancements.

2.       Define and develop new business applications to support dealer, supplier or internal users.

3.       Support existing web applications

4.       Validates all changes to be made to the system through system testing.

5.       Maintains the system test environments with the proper data.

6.       Follow Sarbanes-Oxley IT policies.

 

 Qualifications:  To perform this job successfully, an individual must be able to perform each essential duty satisfactorily. The requirements listed below are representative of the knowledge, skill, and/or ability required.

1.       Excellent working knowledge of Java, Javascript, CSS, HTML, XML

2.       Familiarity with SAP ABAP+

3.       Familiarity with C#.NET+

4.       Good working knowledge of SQL Server

5.       Required to learn how the company Enterprise Resource Planning (ERP) system works and be able to use it in day-to-day activity within functional area.

6.       Required to use the company’s ERP system for creation of SAP interfaces using the ABAP programming language.

7.       Use PC applications such as Word Processors, Spreadsheet/Database programs.

8.       Must have a working knowledge of functional areas they support.

9.       Must have a working knowledge of the global business processes of the company.

10.   Must have good communications with external experts and functional experts within internal departments.

 

Education and/or Experience:

1.       4-year degree or equivalent experience in related functional area

2.       3-4 years experience developing java applications
