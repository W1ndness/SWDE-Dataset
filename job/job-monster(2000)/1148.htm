Web Developer - ASP/SQL - Perm!

About the Job

Our Burnsville client is seeking to add a permanent web developer to their staff.  This is perm/fulltime and is only open to minnesota candidates at this time. 

 

No sponsorships available.

 

Position Summary 

Our Company is seeking a web developer to create, modify and support our dynamic, database-driven websites and eCommunications using ASP, MS-SQL, JavaScript, XML, and CSS.  This position is responsible for coordinating the development of all web projects within the America’s regional sales organization (RSO) supporting the following areas:  Video, Intrusion, Fire and Access Control for Security, ElectroVoice, Sports Communications, Aviation, Dispatch.  The individual in this role will execute the strategic web and eCommunications plan that is rolled out by the Senior Web Developer and use feedback analyses from internal and external customers to improve web presence and execution. 

 

 

Essential Functions: 

 

Web Programming 

·         Develop front-end pages and applications using XHTML, CSS, JavaScript (including AJAX), and Flash. 

·         Develop back-end applications using ASP, and MS-SQL 

·         Provide technical support to an application once it is running, making further adaptations or patches to the code, or creating new versions as needed. 

·         Solve problems by re-writing the code or adding new code to resolve programming issues. 

·         Run tests of applications to identify bugs which need to be rectified. 

 

Design 

·         Work closely with graphic designers and other creative specialists to understand the design concept and advising on how it can be implemented technically within constraints. 

·         Work with the Marketing department to visualize, design and develop Flash/rich media applications to support the Marketing Communications plan (includes product launches and portfolio support). 

·         Sort out operational logic and business rules that are necessary for the feature set to be reproduced correctly according to the designer's specification. 

·         Build and maintain search engine optimization (including possible Google keyword ad campaigns) and measure effectiveness. 

·         Partner with internal/external customers to better understand their needs and online experience. 

  

Requirements 

 

Skills 

·        Intermediate to advanced skills with XHTML, CSS, ASP, JavaScript, and Microsoft SQL. Ideal candidate will have 3-5 years of experience. 

·        Adobe Flash skill set should include ActionScript (at least v2), Flex technology, database and XML integration.   

·        Experience with maintaining Microsoft Content Management Server, Microsoft SQL Server 2000/2005 databases, VB.NET/C#.NET, Adobe ColdFusion, Adobe Flash preferred. 

·         Ability to manage domain names and secure certificates. 

·         Experience with analytics tracking and producing reports that highlight key initiatives. 

·         Ability to research and keep abreast of emerging technologies is a must in order to be able to deliver the most up-to-date solutions such as learning new programming languages or technologies. 

·      Ideal candidates should be capable of fulfilling many project requirements simultaneously as well as experience managing external vendors to achieve results. 

·       This individual must be self-motivated with strong organizational and communication skills. 

 

Education 

·         Bachelor Degree in Web Design and Development or other similar discipline required 

·         Microsoft Developer Certification preferred 

 

Experience 

·         Professional degree coupled with 3-5 years experience
