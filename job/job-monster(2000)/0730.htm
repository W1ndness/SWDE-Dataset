About the Job

Our client, located in downtown Cleveland, Ohio, has an immediate need for an experienced SQL DBA candidate for a 3-month contract to hire opportunity.  Details follow:

Position Responsibilities:

  • Monitor system for normal operations including database availability, performance tuning, indexing, data and error checking and error handling.
  • Recommend upgrades to maintain reliability and performance.
  • Daily administration of database event logs.
  • Follow system management standards and procedures including naming conventions and directory structures, local and remote access rights, server and client software.
  • Perform day-to-day database administration activities to ensure integrity, security and availability of production and development environments.
  • Adhere to standard change control procedures across production and development environments.
  • Make recommendations to improve database and application design standards that will increase performance, reliability, and ease of administration.
  • Work closely with senior database staff in support of project deliverables.
  • Responsible for database access security in adherence with security policies.
  • Work with the Infrastructure team to maintain current releases of the operating system, application and software patch levels.
  • Creation of scheduled processes (ETL, SSIS packages, scripts, etc.) to automate functions, processes and to monitor database usage.
  • Monitor and resolve assigned priority issues in accordance to departmental metrics and established Service Level Agreement (SLA).
  • Work with System Administrators to monitor capacity thresholds including disk and memory storage requirements, assist with setup and periodic failover testing of high availability databases.
  • Work with the Application Development team in support of new applications development and database designs that are efficient and optimized to meet the performance needs of the business.
  • Resolve database issues with support from the Applications Development, Infrastructure Team and vendor support teams.

 

 

Other duties as assigned.

 

Position Qualifications:

 

  • Requires a four-year degree in Computer Science or related field.
  • Candidates must have 5+ years of experience working as a DBA in a software development team responsible for maintaining a Microsoft SQL server environment, data schemas and writing stored procedures.
  • DB2 UDB for iSeries experience a plus.
  • Skills in data aggregation using SSIS; experience with ETL applications.
  • SQL 2000 / 2005 replication and knowledge of high availability environments a plus.
  • Able to work flexible schedules including 3rd shift rotation.
  • Familiarity in Data Warehouse data management.
  • Strong communication skills.
  • Capability in working both independently and as a member of a team.
  • Ability to work in a fast-paced high growth environment.
  • Must have excellent Customer Service attitude and focus.
  • Experience with SQL 2000 / 2005
  • Experience with SQL 2008 a plus


Please note that sponsorship is not available for this position.  Please no visa candidates or third parties.
