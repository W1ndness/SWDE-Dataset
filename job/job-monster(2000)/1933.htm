Network/Systems Administrator

About the Job

DESCRIPTION: 
• Installation, configuration and troubleshooting of server software and hardware. Software includes, but not limited to: Windows(all current versions), Linux, Exchange (all versions), SQL (all versions), IIS, Active Directory, TCP/IP, Wins, DNS, DHCP, VPN site to site, Symantec Anti-Virus, Juniper Switches and routers, Cisco Routers and  Firewalls, MS Operation manager, MS Configuration manager, server updates/patches and Network Security. Backup and restoration of networked servers.
• Installation, configuration and troubleshooting of desktop software and hardware. Software includes, but not limited to: Windows (all current versions), Symantec Anti-Virus, Office (all current versions), and custom designed software.
• Monitor the system configurations ensure data integrity and throughput.
• Management of bulk email services includes, but not limited to: assuring emails are sent, bounce backs are properly distributed, imports and exports of email addresses occur at the designated time and are properly saved.
• PCI-DSS compliant company must go through compliancy and audit processes.
• Assists the Product Support Group employees on items they need to escalate
• Work with the programming department in the development, installation and troubleshooting of custom designed software. Ability to find solutions to current problems to increase productivity in systems.
• Complete projects within designated time frames. Assists the Product Support Group employees on items they need to escalate.
• Escalate items based on the Product Support Group escalation procedures.
• On-Call rotation.
• Up to 20% Travel.
• Other duties as assigned.
• Supervised by Network/System Supervisor
• Ability to Lift 80 lbs. above your head for a sustain periods of time.

MINIMUM REQUIREMENTS (EDUCATION/SKILLS/EXPERIENCE):
• MCSE equivalent skills even if candidate has not completed the certifications; 2+ years’ experience building server class systems and prior experience in a Senior or Lead technical role.
• MUST have at least 2 years of Microsoft operating system experience including experience with Active Directory, 2003 / 2008 servers,, Microsoft Exchange, DNS, SMTP/POP3, SNMP, Microsoft SQL 2000 2005, remote server administration, and proxy servers.
• MUST have continuous, recent day-to-day technical customer support experience in a corporate IT environment.
• Customer Service, Microsoft System Center, Juniper routers and switches, Cisco Firewall and routers, Microsoft IIS, Microsoft SQL, Microsoft Windows all current versions, Microsoft Exchange, Snort, Linux/BSD
