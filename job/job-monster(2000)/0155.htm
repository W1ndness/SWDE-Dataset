Software Engineer

About the Job

 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 

TransCore, a subsidiary of Roper Industries, Inc. (NYSE: ROP), is seeking a full-time Software Engineer to join the IT Team in Houston, TX.   


The Software Engineer is responsible for maintenance and development of C# frontend and backend applications, along with SQL Server development.  This .Net Applications Developer ensures the stable operation of the in-house software application. Working in a team-based Microsoft environment, this professional will be responsible for applying technical solutions to business problems and needs as identified by our customers.  In this role, the developer will be responsible for troubleshooting existing applications and new installations, as well as interfacing with the client.  

 

Duties and Responsibilities include, but are not limited to:

 

  • .NET framework development
  • Strong C# development of both front end and back end applications in a Windows operating system environment
  • Database design work in SQL Server 2005 or 2008
  • Technical writing/documentation to include proposal responses, requirements documentation, Software Design Documentation, User Guides and manuals
  • Performing requirements analysis in an Agile development environment while working with users, 3rd party representatives, project teams and management
  • Work with DBAs and senior team members to develop database designs and enhancements
  • Provide technical assistance by responding to inquiries regarding errors, problems or questions.
  • Train end users or technical support staff as required
  • Analyze, install and test upgrades of products
  • Monitor performance of applications and/or programs after implementation, taking action as required

 

The ideal candidate will have the following skills and experience:

 

  • 3+ years professional experience developing in a .Net environment utilizing C# development skills
  • Strong technical writing/documentation ability
  • Experience utilizing SQL Server 2005 or 2008 (SQL, T-SQL, Stored Procedures) 
  • Web Services and XML experience
  • Service-oriented and system documentation
  • Object oriented design and implementation
  • Code Release Management in the .NET framework
  • Experience with .NET deployment strategies
  • HTML, JavaScript, VB Script, Visual Basic.Net, ASP.NET, ASP, XSL and  Crystal Reporting
  • Posses the ability to propose innovative solutions and communicate ideas to management and to the group
  • Able to work as a team member and be a self-starter
  • Excellent verbal, phone, and written communication skills
  • Additional beneficial skills include:  Oracle, Reporting Services, Java, VB OOAD, COM, DCOM, ActiveX, COM+, MAPI, MS Access, ADO and MySQL

 

Education Requirements:  BS Degree in Computer Science or Software Engineering (or similar) or equivalent experience or certifications.

 

About TransCore:

 

TransCore is dedicated to making surface transportation more efficient through innovation.  With installations in 44 countries,more than 100 patents, and pioneering applications of RFID and satellite communications technologies, TransCore's expertise is unparalleled.
