Research Software Engineer

About the Job

 
 
 
 
 

Implement an advanced machine-learning software package to support cutting-edge research. The prospective candidate will be responsible for all aspects of the package development: conceptualization, design, implementation, documentation, testing, debugging, external software integration and user-support. You will work with a group of motivated and friendly researchers to incorporate their work into the package. This position would serve as an ideal preparation for entry into a CS graduate program.


Mandatory Requirements:

  • Eagerness to learn new concepts

  • Proven object-oriented programming experience (Java or C++)

  • Ability to code data structures beyond the standard ones provided in common software libraries

  • Ability to think analytically about code efficiency, scalability and maintainability

  • B.S / M.S in Computer Science or related discipline


Preferred Qualifications:

  • Experience in implementing machine learning algorithms

  • Good scripting skills (e.g., Perl, Shell and Python)

  • Good knowledge of open-source Java and C++ tools

  • Familiarity with:

    • SQL and relational databases

    • Linux operating system and its tools

    • Ontological schemas such as OWL, OWL2, etc.

    • Numerical optimization concepts


Soft Skills:

  • Strong communication and people skills

  • Ability to interact in a peer-to-peer environment rather than a hierarchical model

  • Ability to work in a busy research environment rather than a 9-to-5 one


Salary: $65,000 + based on qualifications and experience.


Deadline: Open until filled. For best consideration, please apply by Dec 1, 2010.


To Apply: Please email linqs-jobs@cs.umd.edu  a sample of your best code and a letter including salary requirements, resume and contact information of three work references:
