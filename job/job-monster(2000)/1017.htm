About the Job

MLS Property Information Network, Inc. (MLSPIN) is a rapidly-growing Multiple Listing Service serving greater Massachusetts and beyond with immediate need for a talented, experienced .NET developer.

Join the team developing our premier MLS software using current Microsoft technologies (.NET Framework 3.5, SQL Server 2005/2008).  MLSPIN offers a wide range of fantastic benefits, including professional training, 401k matching, health & dental plans, disability & life insurance, exceptional paid time off, and much more. 

We offer a small team atmosphere where your contributions are recognized and rewarded!

Responsibilities for this position:

  • Design and program web based business applications using VB .NET and C#.
  • Design and implement web-services within the middle tier of the application framework.
  • Technical problem solving, programming and unit testing skills.
  • Modify and help shape technical documents with minimal guidance.
  • Solve moderate to complex problems with minimal guidance using appropriate toolset.
  • Provides accurate time estimates for tasks.
  • Design and develop systems using with Web 2.0 principals
  • Design and build Microsoft SQL Server databases which include creating tables, writing stored procedures, queries, triggers, and cursors.
  • Design and build Microsoft Reporting Services solutions as a stand-alone application or integrated into new or existing web applications.
  • Document, troubleshoot and isolate moderate to complex problems encountered during unit testing.
  • Work with other members of the Application Development team to help coach others, improve our processes, effectiveness and efficiency.
  • Creates and executes project work plans and revises as appropriate to meet changing needs and requirements
  • Communicates and enforces coding standards, perform code reviews
  • Consistently delivers high-quality services and products.

 

Required Qualifications for this position:

At least 5 years’ experience developing software in a Microsoft environment (VB.NET, SQL Server), with demonstrable understanding of the .NET Framework including VB.NET, ADO.NET, SQL Server, collections, caching, etc.

At least 5 years’ experience developing n-Tier applications.

At least 5 years’ experience with hands-on SQL Server development (T-SQL queries, Stored Procedures, Views).

Practical experience with agile development and software design concepts.

Written and verbal English skills.

 

The ideal candidate will also have experience with some of the following:

ASP.NET in VB.NET

Classic ASP knowledge is helpful

HTML, CSS, JavaScript

AJAX with .NET Web Services

XML/XSLT

Template-driven systems, ASP.NET Themes & Skins

Working in n-Tier architectures

Experience with Microsoft Team Foundation Server helpful
