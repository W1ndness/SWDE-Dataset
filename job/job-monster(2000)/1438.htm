MS Exchange - Programmer/Analyst/Designer - Senior

About the Job

Description
Support the architectural design and implementation of the MS Exchange 2007 infrastructure. Design and implement the conversion from Lotus Notes / Domino email system to MS Exchange 2007. Provide knowledge management documentation for administration processes and procedures Provide detailed Lotus Notes/Domino Migration Plan to include standard operating procedures for migration. The migration plan will define the processes and dependencies for the migration and should include communication requirements for users and clear instructions on new mail use, SharePoint, Instant Messaging and Live Meeting functionality. Execute successful E-mail migration – 42,000+ mailboxes based on the approved schedule. Provide detailed application migration plan (for simple Domino databases) to MS SharePoint. The plan should include standard operating procedures for application migration and define how the contractor will make sure that user functionality and application performance is not impacted by the migration. Execute successful Application Migration (simple Domino applications only)
Migration plan to include target state technology definition for complex applications to existing FS tools and/or technology.
Education
Bachelor’s degree, specialized training, or equivalent work experience.
Experience
Typically requires 5-7 years of relevant information technology work experience.
Requires a high level of expertise in multiple system environments. Including the following:
Experience in developing, implementing and supporting Exchange applications in a large (36,000 employees) and regionally dispersed organization. Microsoft Certified Master in Exchange Server 2007. Must have a thorough understanding of Microsoft Exchange Server design and architecture., MS SharePoint, LiveMeeting and Instant Messaging, Active Directory, Domain Name System and other core network services related to Exchange Server. One or more years of hands-on experience with Microsoft Exchange Server 2007: installing, configuring, and troubleshooting. Ability to work virtually and independently, as well as directly with the FS virtual administration team and end users.
Special Requirements/Certifications
Candidates must be able to pass a background check.
