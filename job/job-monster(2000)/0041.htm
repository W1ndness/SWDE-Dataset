Database Administrator

About the Job

Wouldn’t it be nice to work for a company where the mission makes sense, the salaries are competitive, the company listens to the needs of the employees and the office is professional yet the dress is casual? 


Colorado Access is a nonprofit health plan serving the medical and behavioral needs of the medically underserved in Colorado.  We offer an excellent benefit and compensation package, including medical, dental & vision, 401(k), tuition reimbursement, and competitive PTO.


We are currently seeking a Database Administrator responsible for all corporate database schema designs, database implementation and administration. The position is also responsible for reviewing and monitoring database applications & performance, support and troubleshooting, as well as working closely with staff and consultants associated with Colorado Access database operations. Establishes standards and procedures concerning the database systems. Administers and controls the Colorado Access data resources. Maintains data dictionary to ensure data integrity. Develops procedures for security and to recover corrupted data. Monitors database performance and utilizes tuning tools/methods to improve database performance.


Essential Functions Summary:

·         Participate in logical/physical database model designs and implement as required.  Develop, implement, test and maintain SQL, PL/SQL, shell and utility scripts.

·         Investigate and resolve database issues by utilizing database monitoring, maintenance and performance tuning techniques. Research, test and implement Oracle patches and upgrades as needed.

·         Instruct and assist development team in the use and application of database features and functionality, database design and suggested best practices for the SDLC process as it pertains to the database environment.  Provide support and mentoring of Jr. level DBA’s and developers as needed.

·         Recommend and perform security enhancements of the database and data.

·         Responsible for establishing, promoting and maintaining database standards and procedures in conjunction with management.

·         Test and maintain reliable backup and restore processes and procedures.  Test and maintain an efficient and workable disaster recovery process for the database platforms.  Ability to provide 24/7 on call rotational support.


Requirements/Experience:

·         Bachelor’s degree in Computer Science or Computer Engineering and ten (10) years progressive work experience in a related field, or any equivalent combination of experience and/or education.

·         Strong knowledge of Oracle 10g/11g database and architecture, RDBMS, Physical/logical database design and implementation,  Oracle Enterprise Manager Application,  coding and troubleshooting PL/SQL,  Unix/Linux and Windows OS and shell scripting in Unix/Linux and Windows OS.

·         Ability to work with developers with regard to normalization of data structures, optimization of SQL queries and promotion of objects thru the SDLC process.

·         Proficient at code management using CVS and/or Subversion.

·         Advanced Oracle development and production DBA experience.

·         Experience with MS SQL Server administration/development, Java and Perl preferred.
