About the Job

VantageLinks is dedicated to Information Technology (IT) staffing through permanent placement, contract-to-hire, and contract services.   We also offer Team Services and Information Systems Development Solutions.  We specialize in finding Information Technology professionals ranging from help desk specialists to programmers including; systems analysts, designers, architects, database professionals, ERP specialists, and project managers.  We support a wide range of environments including but not limited to .NET, Java, UNIX. For additional information and complete list of openings, please visit our web site at www.vantagelinks.com .

 

We are looking for a SQL Server DBA

 

Experience Required:

·         5+ Years of experience

·        SQL Server 2005 and SQL Server 2008

·         Ability to work in a team and individually

·         Ability to work in a 24/7 environment

·          SQL programming, T-SQL, triggers, and stored procedures

·         Experience working in a development and production virtual environment

·         Experience with mirroring, replication, and performance tuning

·          Experience with clustering, backup and recovery

·          Previous experience in database upgrades

·         Enhancing SQL Server database performance, stored procedure performance tuning, and SSIS performance tuning.

·         Participate in and contribute to the implementation of best industry practices for Microsoft BI tools and technology.

·         Contribute to SQL Server architecture and capacity planning activities

·         Work with the business users to resolve support and technical questions

·         Change management and migration of changes from environment to environment

·         Contribute to best practices for managing the SQL Server data warehouse environment

·         Monitor the Microsoft SQL Server data warehouse environment for resource utilization and act on incidents in conjunction with established service level agreements

·         Prefer experience to include BizTalk and SSRS (SQL Server Reporting Services)

·         Prefer a four year degree in Computer Science, Engineering, or related discipline

 

Roles & Responsibilities:

·          Responsible for database server upgrades from SQL Server 2005 to SQL Server 2008

·         Provide 24x7 support and maintenance for production databases

·         Maintain and monitor database backups and rotating archives of backup files

·         Production database disaster recovery

·          Configure and monitor database mirroring for mission-critical databases

·          Configure and monitor log shipping for selected databases

·          Daily database server health checks for: backups, jobs, mirroring, replication, etc.

·          Monitor and profile production database servers for load and performance bottlenecks

·          Query optimization and performance tuning

·          Assist application developers with database schema and procedure changes according to company conventions and best practices

·          Analysis and approval of database schema and procedure changes from dev to staging to production environments

·          Write and update advanced stored procedures for business reporting purposes

 

Secondary Responsibilities

·         D BI and report creation using SSRS
