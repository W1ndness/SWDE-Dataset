About the Job

Job Description:

 

Maximize Technologies LLC is currently seeking a Software Developer to join the Maximize Team.  This is a full-time salaried position with benefits. 

 

Maximize Technologies has some exciting new projects that need attention; Web Application Development, E-Commerce Website Development, Website Re-Design Development, PHP MySql Development and potential Google Apps and Cell Phone development.

 

We are searching for a person who is responsible, mature, organized, dedicated, hardworking, able to multi-task, prioritize, work on their own, flexible and be responsive to many different client requests.   

 

Responsibilities:

 

  • SDLC (System Development Life Cycle)
  • .Net Web Application Development
  • SQL Stored procedure Development
  • Database Design (Entity Relationship Modeling)
  • Website Development (PHP, MySQL, HTML, XML, etc...)
  • Different programming protocols (depending on client needs)
  • Project Status Reporting
