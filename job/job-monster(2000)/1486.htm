Integration Engineer

About the Job

 

Our client located in Dulles, VA is currently looking for Integration Engineers to join the software development group to work on a ground break, company-wide new initiative. The job requirements and responsibilities for this position are the following:

1.       1-5 years Java development and/or testing experience

2.        Interface with the 3rd party organizations/customer (ie: Gap, Best Buy, etc), multiple internal and external constuients including development, QA, UI and design, content and editorial staff, security, etc.

3.       Define, build, document, test and deploy APIs using Java JDK, and JSON to interface between AOL’s and 3rd party customers’ standard billing, customer data, and other necessary integration points required for internet purchasing.

4.       Exceptional communications skills are required

5.       Strong sense of urgency, ability to prioritize and manage multiple concurrent, changing, and competing priorities, at any given time.

6.       Troubleshoot issues that occur with the various retailers to ensure a seamless process to the end consumer.

7.       Exceptional customer orientation, attention to detail, flexible and a “whatever it takes” attitude to make sure each milestone of the project is kicked off on time.

8.       E-commerce industry experience is highly desired

9.       Technical skills:

a.       Java, JSON, JDK
