Healthcare Database Analyst

Jeffersonville, IN

(Job ID: 1666)

 

We are seeking an experienced Database Analyst with in-depth knowledge of healthcare enrollment systems. Primary responsibilities include the following:

  • Database schema design and implementation, data importing, manipulation, and reporting using MySQL
  • Interpretation of complex data requirements dealing with health care claims and enrollment data
  • Implementation of import/export scripts to automate data transfers

 

The ideal candidate will be able to work in an ever-changing environment, easily moving from task to task while maintaining overall productivity while administering some or all of the following backup/secondary responsibilities:

  • End user support and troubleshooting database/web application issues
  • Database schema, script, application, and process documentation
  • Data mining and operational support

 

Required Skills:

  • 2+ years database experience (MySQL preferred)
  • 1+ years healthcare experience

 

Required Experience:

  • SQL (MySQL)
  • Shell scripting using PHP, Perl, or similar language
