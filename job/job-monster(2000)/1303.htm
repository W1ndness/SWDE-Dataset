Staff Applications Developer

About the Job

Beckman Coulter develops, manufactures and markets products that simplify, automate and innovate complex biomedical testing. Our diagnostic systems are found in hospitals and other critical care settings around the world and produce information used by physicians to diagnose disease, make treatment decisions and monitor patients. Instruments for life science research are used by scientists as they study complex biological problems including the causes of disease, identify new therapies, and test new drugs. Our customers include hospitals, physicians' offices, diagnostic reference laboratories, pharmaceutical and biotechnology companies, universities, medical schools and research institutions. In fact, Beckman Coulter has more than 200,000 clinical and research instrument systems operating in laboratories around the world.

Job Description:

The Staff Applications Developer provides technical expertise on one or more applications/or development tools and serves as a technical consultant to system applications group. Incumbent provides technical consulting support on projects or system issues. The individual evaluates, tests, and consults on pilots to ensure that system application meets customer expectations. These individuals analyze existing programs, formulate logic for new systems, devise logic procedures, prepare flow charting, code, document and test/debug programs. They are responsible for the technical implementation and upgrades of major systems, particularly large vendor provided applications. In addition, the Staff Applications Developer provides technical direction, quality assurance, training and guidance to system application teams. May work on system performance recommendations.

Requirements:

A candidate must have: Engineering experience in a large web environment. Hands on experience in Java SE/J2EE/JEE development. Hands on experience in designing and implementing 32 and/or 64 bit C++ (MSVC++6, MSVC++ 2010) applications on Windows XP, 7, 2008 and Vista platforms. Experience with C++ multi threading, pointers, data structures, blocking queues, sockets, I/O, XML. Experience in profiling C++ applications and testing for memory leaks, runaway threads, CPU pegging, race conditions, and other things that impact C++ performance and/or memory usage. Knowledge of C++ unit testing framework is a must, especially if a framework can support both MSVC++6 and MSVC++2008/2010. Knowledge of "google test" testing framework is a plus. Experience with WebLogic Application Server. Experience in creating applications for multi-node Oracle RAC clusters. Experience in creating UML2 architectural designs. Experience in working with very large datasets. Experience in JEE5+/J2EE programming, JDK 6, Java Servlets API, JSP, SOAP, multi-threading, XML, XSLT, JDBC, IPC, Oracle SQL, Oracle Stored Procedures, secure communications over HTTPS and sockets, EJB2/3, JMS, JNDI, JTA, JCA, JDBC, JMX, JAAS, Security API, Weblogic Portal API, Shell Bash Scripting on Linux, ANT Scripting. Experience in UI technologies: GWT, Weblogic/Apache Beehive and Java Server Faces. Successful history of introducing and/or implementing new technologies and/or practices. Experience in optimizing code, configurations or the infrastructure to improve scalability of the application/system. Extensive knowledge of Red Hat Linux platform including IPTables, networking, repository management, remote upgrades, creating RPMs, optimizing JVM and Java applications for computers with limited resources, scripting using Bash, Perl and PHP. Experience with Java Key Store. Experience in Test Driven Development, Unit testing, Continuous integration systems, Static analysis of Java code, JVM profiling. Experience in leading the entire software application development lifecycle including designing, implementing, testing and maintaining the code. Knowledge of object-oriented analysis and design. Knowledge of design patterns. Knowledge of advanced data structures, algorithms, and multi-threading. Ability to understand and use business requirements and create application designs to meet those requirements while adhering to non functional qualities of the architecture. Ability to lead an offshore remote team in implementing and delivering the software based on design specifications. Desire to train, mentor and assist offshore team members who are tasked with design implementation. Excellent interpersonal and communication skills. Commitment to the project and desire to succeed. Flexibility to adjust the work schedule to minimize project risks due to different time zones between the US and offshore teams. B.S. degree in Computer Science with 7+ years of relevant experience, or Masters degree in Computer Science (or Software Engineering) with 5+ years of relevant experience.
