About the Job

Named one of the “50 Best Places to Work” in Massachusetts by the Boston Business Journal for 2008 and Named to The Boston Globe 100 Top Places to Work '08, TechTarget produces the most targeted media for enterprise information technology (IT) professionals, including invitation-only conferences; controlled, qualified e-newsletters; Webcasts; and the TechTarget network of Web sites targeting specific sectors of the IT market. TechTarget leads the industry in developing innovative advertising vehicles that address IT marketers' need for high-ROI branding and lead-generation programs. The leading IT companies in the world rely on TechTarget media to reach qualified IT professionals with buying power. TechTarget has more than 1,500 advertisers, including IBM, Microsoft, Oracle, HP, Cisco, Intel, and EMC. More information about TechTarget is available at www.techtarget.com

If you’re a self-starter who thrives in a fast-paced environment, TechTarget has the following opportunity:  Software Engineer.

Software Engineer to join a team responsible for the design, development and implementation of Java-based web applications and scripts.  The applications are database-driven (MySQL/Oracle), use Hibernate for ORM, Spring Framework, and Struts2.  This is a Java Application Development position, requiring proficiency at all levels (database, Java middle-tier, and view layer), but successful candidate will show a solid understanding of the database access layer, the database level itself, and be comfortable working in database code (modeling, optimizing, and SQL scripts for data definition and manipulation).  Hibernate experience is required.  Additionally, position requires an excellent understanding of the full software lifecycle, including functional requirements gathering, technical documentation and design, development, testing, building, and deploying to production.  Experience with an agile methodology is desired.  This is an excellent opportunity for a Java application developer with 4+ years professional application development experience to step into an established team and drive design and development of critical systems as well as to grow knowledge and experience by working with a large group of experienced developers.

Essential Job Functions:  

·         Java web application development

·         Java database access development

·         Data-modeling and database design skills

·         Database programming

·         View/client-side development

·         Interaction and cooperation with business owners, enterprise and system architects, developers, project managers and QA Technical documentation

Knowledge, skills and abilities required:

·         BS in Computer Science or related engineering or scientific program

·         4+ years total database-driven  web application and script development experience using  Struts2 or Spring MVC

·         3+ years Java

·         1+ year Hibernate

·         3+ years database application development experience:  JDBC, ORM, data modeling, SQL, Oracle PL/SQL

·         Experience with at least one web application server (such as JBoss or Resin)

·         Solid understanding of and experience with web request/response concepts - cookies/domains/caching, etc...

·         Candidate must be self-motivated, innovative, and able to work independently and within a team in a fast paced environment

·         Pluses:  Multiple ORMs, Struts, SOAP, REST,AJAX, MySQL
