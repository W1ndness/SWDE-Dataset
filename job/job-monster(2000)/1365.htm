JR System Programmer / IT Support

About the Job

JR System Programmer  / IT Support Job ID: 207-MN

 

Job Description 

THIS IS A GREAT START FOR ANY PROGRAMMING CAREER! Luxury Mortgage has career opportunities for JR System Programmer!

Daily responsibilities include developing tools within our loan system utilizing basic programming knowledge including but no limited to developing custom input screens, custom print merge forms, building system rules, triggers within VB Script and .NET language.  Develop reporting tools as directed by management using Microsoft Excel, PHP and mySQL.  

 

Manage, modify and update website as directed by management. Develop and manage various API’s with various vendors.

Position available in Stamford, CT location - No Telecommuting - must work all hours on-site.

Job Requirements 

Minimum Qualifications:  NO PRIOR WORK EXPERIENCE REQUIRED BUT MUST HAVE KNOWLEDGE

 

Candidates must possess in-depth knowledge of Microsoft desktop operating systems, experience performing intermediate network administration, TCP/IP networking, basic PC/server repair, experience managing Windows 2000 & 2003 servers, active directory, DNS functions and have a proven ability to quickly learn and troubleshoot other technologies including: Exchange, Citrix server, MySQL and other.

Job experience/qualifications required 

  • THIS IS A PROGRAMMING ENVIRONMENT
  • Ability to work in a fast-paced, high volume environment, while maintaining time lines 
  • Intermediate to advanced  in VB Script, PHP and .Net programming
  • Must have excellent problem solving and math skills
  • mySQL database construction/​integration
  • CMS construction/​design
  • jQuery
  • CSS knowledge/​implementation
  • Knowledge and experience working with SDK’s and API’s
  • Demonstrated verbal and written communication skills with the ability to explain features, policies and benefits to business partners.


 additional skills PREFERRED by not required 

  • Experience with Ellie Mae Encompass 360 Banker addition
  • Basic knowledge of  security groups and user privileges
  • Basic knowledge of level 1 and 2 customer service network /systems issues
  • Basic knowledge of administering and managing network profiles within Active Directory of Windows server environment
  • Basic knowledge of conditioning and deploying equipment as needed
  • managing/addressing user level trouble tickets both locally and remotely, maintaining equipment inventory, system upgrades and staff and branch on-boarding. 

 

Salary $38,000 plus benefits.
