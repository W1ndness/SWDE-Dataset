Network Security Architect

About the Job

Link Technologies is currently looking for a Network Security Architect for an exciting opportunity in Las Vegas, Nevada. The Network Security Architect is responsible for company’s networked and security systems.  They will help design, implement and support secure system architectures in a large ever changing environment.  Responsibilities will include being a security centric subject matter expert on the many systems creating a secure and regulated environment.   The Network Security Architect will constantly mentor the many teams throughout IT on secure practices.  Architected solutions will require clear strategic vision and the ability to define a roadmap to success.  Will create and use accurate risk assessments to employ compensating controls to ensure a business enabling secure solution is deployed.  Secure solutions will inherently run parallel to IT Operations and business processes focusing on self documenting auditable procedures. If you are interested in working in a team environment with a growing company, we are interested in talking with you.

 

 Responsibilities Include:

·         Develop, maintain and review technical drawings and application documentation.

·         Validate as built application data flows and security controls are functioning.

·         Certify, document and test server and firewall appliances.

·         Review and write standards, procedures and controls as necessary.

·         Evaluate and prioritize security risks.

·         Assess security of hosts and networks.

·         Make decisions for functional areas in normal and emergency situations.

·         Execute strategic and tactical priorities for Network Intelligence program development.

·         Effective within SIEM console.

·         Develop automated data push processes to feed Network Intelligence into IT Operations to help diagnosis and resolve complex problems disrupting the business.

·         Ensure required systems are reporting necessary compliance telemetry to ensure PCI, SOX and MICS reports are complete.

·         Lead asset management practices for IT Security organization including:

§  Preparing the internal network for network admission controls.

§  Ensuring network endpoint profiling solution is effective.

·         Maintain security solutions in accordance with policies and control standards.

·         Manage Security posture as it relates to Operating System risks for Networking routers, switches, firewalls, and miscellaneous appliances.

·         Insure Network related standards are developed and maintained.

·         Manage networking software versioning, updates, policies and procedures.

·         Manage information security risks to an acceptable level and identify appropriate compensating controls to mitigate security risks to the business solutions.

·         Provide project management for security initiatives involving server, software, operating system and network upgrades and installation.

·         Ensure adherence to PCI, SOX and NGCB MICS standards and guidelines for security, data integrity, data and system availability.

·         Work with internal and external auditors to maintain SOX, PCI, MICS and other internal and external compliance.

·         Responsible for identifying information security risk within the enterprise and setting priorities for risk mitigation efforts.  Provides oversight to ensure technical compliance with security related regulatory requirements (PCI, SOX, NGCB MICS, etc). 

·         Manage security related aspects of all remote access mechanisms for remote users.
