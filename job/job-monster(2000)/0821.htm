About the Job

PHP/MySQL Guru Wanted!

Do you love developing cutting-edge web applications? Are you a PHP guru? Do you want to be on a team of developers building the next generation product from an existing market leader?

If you've answered "Yes" to these questions, eMaint Enterprises has a great opportunity for you.  eMaint Enterprises is looking for enthusiastic, bright, motivated developers to join our PHP development team.  The team is currently working on the next generation of our Enterprise Asset Management and Computerized Maintenance Management System, a web-based application that uses Linux, PHP, MySQL, and the Zend Framework targeting desktop browsers and mobile platforms.  

Developer must haves:
    • 3+ years experience in building PHP / MySQL web applications.
    • Team player and willing to work with others for the good of the project and the team.  This means working on code that others have written, having others work on your previously written code, implementing your own code solution or implementing another co-worker*s code solution, all with a good attitude.
    • Can program in PHP using good form, using standard approaches to solve problems.
    • Can program using object-oriented approaches.
    • Can program in Javascript (using a framework is usually satisfactory).
    • Can create/edit/update MySQL databases/tables in a variety of ways (using a GUI or within MySQL command line interface ).
    • Unix / Linux familiarity  (can change directories, change permissions on files, move and copy and create files, can ssh, start up mysql client).
    • Aware of security risks involved in web programming, specifically PHP, and takes strides to develop code that is secure.
    • Expert at HTML/CSS (if given some time, they could mock up the home page of amazon.com as a rough draft).
    • Legally authorized to work in US.
    • Local to Marlton, NJ (no telecommuting available).


Developer nice to haves:
    • Familiarity with the Zend Framework.
    • Familiarity with source code control in Unix / Linux (like subversion, git, or CVS, we use subversion).
    • Experience with ExtJS javascript library (or another JS framework--we use ExtJS).
    • Can sys admin Unix / Linux servers (Apache, PHP, MySql, Memcached, Subversion).
    • Familiarity in programming with Memcached.
    • Worked with web applications in high availability/high scalability scenarios.


Note: When applying, please include salary requirements and any sample code you may have to demonstrate your abilities. 


About the Company:
eMaint Enterprises (http://www.emaint.com), located in Marlton, NJ (USA), is a small but energetic, growing software company looking for enthusiastic new team players to help achieve its goals. We are a dynamic company that uses leading-edge technology to develop business-to-business products and services. Our products are backed by people seeking to provide the best service possible. Recent market growth has presented an excellent opportunity for our company and team members to prosper within the areas of Enterprise Asset Management Software and Computerized Maintenance Management Systems. eMaint Enterprises offers a unique environment that fosters individual growth and rewards performance. You'll be surrounded by people who are passionate about what they do.

eMaint Enterprises is an Equal Opportunity Employer. Local applicants encouraged to apply. Smoke-free workplace. Drug-free work environment. All resumes are held in confidence.
