Senior Java Web Developer

About the Job

We are looking for a Senior Java Web Developer to build best-of-breed products that combine internally and externally build applications for a rich user experience on a large B2C and B2B web network.

 

Specifically, he/she will:

·       Develop and maintain web applications in Java that use JSPs, Spring MVC, and Hibernate connecting to PostgreSQL and MySQL.

·       Integrate applications into various content management systems and decorators including OpenCMS.

·       Integrate applications with display-layer technologies including Javascript, AJAX.

 

Requirements

 

·       Enthusiastic developer with a team-oriented development philosophy.

·       Excited by new technologies and embracing projects that “reduce the number of lines of code”.  

·       Experience working in a small group of developers in a fast-paced environment.

·       Experience working with various open source applications and integrating them into an enterprise framework.

·       Experience with large-scale content management systems.

·       Experience with Java and Spring MVC in a Unix/Linux development environment.

·       3-5 years of web application development in a B2C or B2B environment.

·       Strong written and verbal communication skills including analytical capabilities, specification gathering/writing, and troubleshooting skills.

·       Highly motivated and results driven in both team and individual projects.

·       Must be able to meet deadlines and work well under pressure.

·       BS/MS in CS or related degree.

 

Bonus Requirements

 

·       Experience with mobile applications

·       Previous start-up experience.

·       Experience with PostgreSQL and MySQL.

·       Experience with e-commerce systems.

·       Experience with OpenCMS.

·       Experience with PHP, Ruby-On-Rails and AJAX.

 

Compensation

The contribution that is expected of the Senior Java Web Developer is reflected in a highly competitive compensation program consisting of base salary, and bonus, along with a comprehensive program of employee benefits.

 

Please email resume to ITJobs12@dlife.com
