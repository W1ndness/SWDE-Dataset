Server Engineer

About the Job

Atos Origin is a global information technology services company.  Our business is turning client vision into results through the application of consulting, systems integration and managed operations. Atos Origin is one of the few companies in the world that can provide all the 'design, build, and operate' elements of a business solution.    

The company’s annual revenues are in of excess $7.3 billion and we employ 50,000 talented and creative people in 40 countries. Atos Origin is the Worldwide Information Technology Partner for the Olympic Games and has a client base of international blue-chip companies across all sectors. Atos Origin is quoted on the Paris Eurolist Market and trades as Atos Origin, Atos Worldline and Atos Consulting. For more information, please visit the company’s web site at www.na.atosorigin.com.

 

Job Description:

The Server Technical Engineer is part of Atos Origin's expert team of server professionals.  He or she will be responsible for the following activities:

  • Coordinates daily server environment operations: moves, adds, changes and preventive maintenance. Coordinates Server hardware and software upgrades, including verification of proper licensing and the IS department vendor liaison. 
  • Promotes standardization of the server environment and the internal network topology.
  • Coordinates internal network integration with local and remote user systems.
  • Coordinates server environment and supporting services with corporate WAN. Troubleshoots server and internal network issues and expedites problem resolution.
  • Participates in the 24 hour 7 day a week support of the Network Operating Center by taking call on a rotational basis with the other server team members.
  • Establishes and implements server environment policies, procedures and standards; ensures their conformance with Information Services.
  • Uses personal transportation (mileage is reimbursed) for daily commute between work assignments. 
  • Must rotate after hours/ weekend on-call as second level technical support. 
  • Ensures that adequate disaster recovery methodologies are implemented. This includes backup of network equipment and servers, restore procedures, routine data integrity checks, offsite storage of backup media, etc.
  • Ensures that internal network security standards are implemented and monitored on a routine basis. Reports may be provided to the IS Security Officer.
  • Provides ongoing, effective and timely communications regarding the status of any compromised device or application they are working on to the IS department and any other affected department.
  • Provides regular reports to their immediate supervisors on the status of their assigned duties and projects.
  • Implements and supports key applications and services such as: server patch management, anti virus, email system, DNS services, dhcp services, ldap services, and backup methodologies. 

 

Job Requirements:

The ideal Candidate will have at least one current Major Certification -- MS Windows Server 2000 or MS Windows Server 2003 preferred (Microsoft MCSE, UNIX / LINUX, Citrix CCEA, CISSP). Additionally, the selected Candidate will have a minimum of 3 years of experience directly administering a Windows / Active Directory system. The administration experience should encompass managing and troubleshooting the following aspects of the system:   

  • Equallogic SAN and/or EMC SAN
  • Citrix
  • Application server
  • ADS
  • Accounts
  • Domain controller
  • Printing services
  • Security / Auditing services
  • LDAP services
  • DNS services
  • DHCP services
  • IIS services
  • Terminal services
  • Group Policies
  • Software distribution
  • MS Patch distribution
  • Antivirus application (preferably McAfee)
  • Backup and Restore (Computer Associates preferred)
  • Performance monitoring and troubleshooting tools
