Job Description 

The Senior Systems Engineer will support the Joint Tactical Radio Set (JTRS) Enterprise Network Manager (JENM) SoftWare Service Support (JENM SwISS) Program. This project seeks to define a common Service Oriented Architecture (SOA) software architecture for Network Management, to refactor the JTRS Wideband Network Waveform (WNW) Network Manager (JWNM) and the Soldier Radio Waveform (SRW) Network Manager (SRWNM) to that architecture, and to support new Govt requirements that will apply to the integrated system. Responsibilities will include analyzing the functional and performance requirements and Use Cases for the management of multi-level, multi-waveform, heterogeneous networks, both IP and non-IP; defining external interfaces including human interfaces; addressing the system, network, and security architecture and design, functional decomposition of the requirements, and developing a set of activity, sequence, and state chart diagrams. The requirements and the Use Cases will be analyzed to develop derived requirements and define the states and modes for JENM. The system design will be documented in a Dynamic Object-Oriented Requirements System (DOORS) database and Rhapsody, Unified Modeling Language (UML) diagrams and several specifications will be generated. Candidate will prepare for and actively participate in Program/Technical Interchange Meetings (PIMs and TIMs) and Preliminary Design Reviews (PDR) and Critical Design Reviews (CDR).
