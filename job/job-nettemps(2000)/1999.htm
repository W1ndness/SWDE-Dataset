Title : Sybase Upgrade & Cross Platform Scheduler (200001-11) Location : Salem, OR Duration : 9+ Months This position is targeting a PM 3 (Project manager 3) The Sybase Upgrade project is part of an overall objective to migrate enterprise wide infrastructure DBMS software to the most recent version. This will enable the agency to leverage optimum vendor support, take advantage of new functionality and enable the applicable systems to implement data encryption as specified in the state wide Clear Text initiative. This project includes the upgrading of the 10K+ PC clients State wide as well as the primary Sybase data bases. The Cross Platform program is part of the SDC Tools Consolidation program to migrate the State to a standard suite of infrastructure software while de-installing older, silo-oriented software. The Cross Platform Software allows for a central job repository and management system to control the job execution on each platform based on events from other platforms. As business applications are being developed on all of the standard platforms (e.g., z/OS, AIX, Windows) this software provides the capability to integrate the various cross-platform jobs which reduces wait time, down time and increases system availability. This project includes the migration of the existing main frame jobs from the existing job scheduler as well as the planning for the job migration on the non main frame platforms.
