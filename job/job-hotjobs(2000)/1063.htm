Job Description

SQL Server DBA - Database Analyst, Performance Tuning, Globalization, Load Testing, SSIS, DTS, Analysis Services

Based in Yardley, PA, we are a very successful Software Engineering firm that specializes in delivering Enterprise Solutions and ERP systems. Due to growth, we are looking to hire for a Sr. SQL Server DBA/Analyst that possesses experience doing database performance tuning. Experience with Load Testing and Globalization would be a huge plus, but not required. If this sounds like you, please apply ASAP for an interview. We are looking to fill this position immediately.

Objective: Staff Database Analyst to develop and maintain SQL Server databases for an ERP system.

Primary Job Responsibilities:
Database development in SQL Server 2000/2005/2008/2008R2 environment. Assist in analyzing, developing, and implementing the physical database design in support of stated project requirements. Create scripts for database schema changes using TSQL. Reviews, evaluates, designs, implements and maintains databases. Work with Development team on new feature set for product release. Provide leadership and mentoring to other DBA and Development team members. Work directly with customer base to help resolve high level performance issues as it relates to the SQL Server database. Analyze future SQL Server versions and provide areas that would be beneficial for our products.

Must Have Skills:
1.) 5+ years development SQL Server DB Administration (some production would be beneficial too). In depth SQL Server (2000 and 2005) experience.

2.) Must have strong database Performance Tuning experience.

3.) B.S. in Computer Science or equivalent degree or a minimum 5 years of directly related experience

Nice to Have Skills:
~Support of SQL server tools including DTS, SSIS, and analysis services.
~Diagnosis and troubleshooting of complex database issues.
~Prior experience globalizing SQL Server for ERP application is a big plus. ~Collaboration with development organization, Microsoft, and other vendors to resolve infrastructure and architecture technical issues.
~In-depth knowledge of information systems concepts. Strong knowledge of Disk I/O, RAID levels, File Placement best practices.

For your hard work, you will be rewarded with an offer that will include an aggressive base salary ($90,000 - $120,000), 2 Weeks of Vacation, full benefits, a great and fun working environment, and other cool perks! Interviews are occurring early next week, so apply now if you are interested. Local candidates preferred.

Must be authorized to work in the United States on a full-time basis for any employer.
