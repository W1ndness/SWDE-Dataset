Job Description

Outstanding C# .NET Software Engineer, C#, ASP, AJAX, SQL needed!

We are looking for a talented Software Developer to work as a key member of our growing software team! This is a great position for the ambitious person that excels in fast paced environments, enjoys rapid / agile application development and has a great 'software can do anything' attitude!

In this position you will have the ability to work on both AJAX enabled web applications and desktop applications, using C# with an SQL Server backend. If this sounds like you please read on!

Required job skills (at least 2 years experience):

- Developing in C# .NET, ASP.NET, Visual Studio
- Knowledge of using jQuery to develop dynamic, interactive web sites
- Development on the Microsoft .NE 4.0 framework
- Experience writing SQL and stored procedures on MS SQL Server 2005/2008

What is in it for you:

- Strong salary and flexible hours.
- Work with the best tools money can buy.
- A work environment where creativity, not micro management, flourishes.
- A great team to work with in a fun environment and with as many free sodas and snacks as you'd like!
- 401k plan with company contributions.
- Full PPO or HMO Healthcare Plan - your choice; company pays 100% of your premium!
- Dental plan (also 100% company paid)

About us:

We are a tech savvy firm based in beautiful Irvine, CA - only a few blocks away from the Irvine Spectrum Center!

If you have 2+ years experience as a C# Programmer - please apply today so we can tell you more about the position.

Must be authorized to work in the United States on a full-time basis for any employer.
