18x Zoom with Wide-Angle / 3" LCD / SD - SDHC Card Slot / Still and HD Movie Modes / Multiple Shooting Menus / HDMI USB and AV / Monaural Sound
While the Finepix S2800 may resemble a DSLR camera, it is designed for anyone to use. Its appearance is due to an 18x Zoom Lens that extends from 28mm wide-angle to 504mm (35mm camera equivalent) super-telephoto.
  • 14.0 million Pixels CCD
  • Storage media Internal Memory (Approx. 23MB) Slot for Optional SD memory card or SDHC memory card
  • Monaural Sound Record and play
  • Still image: Up to (4:3) 4288 x 3216 / (3:2) 4288 x 2864 / (16:9) 4288 x 2416
  • 18x Optical zoom lens f=5.0 - 90.0 mm, equivalent to 28 - 504mm on a 35 mm camera
  • Digital zoom Approx. 6.7x
  • Multiple Focus modes including Macro
  • ISO Sensitivity Auto / Auto(400) / Auto(800) / Equivalent to ISO 64 / 100 / 200 / 400 / 800 / 1600 / 3200* / 6400* (Standard Output Sensitivity)
  • Exposure control TTL 256-zones metering; Spot, Multi, Average / Exposure mode Programmed AE, Shutter Priority AE, Apeture Priority AE, Manual
  • Shooting modes - Natural Light, Natural Light & with Flash, Zoom Bracketing, Smile, Portrait, Landscape, Sport, Night, Night (Tripod), Fireworks, Sunset, Snow, Beach, Party, Flower, Text
  • Focus Mode: Single AF, continuous AF / Type: TTL contrast AF, AF assist illuminator available / AF frame selection: Center, Multi, Area, and Tracking
  • Multimode pop-up electronic-flash
  • Flash modes Red-eye removal OFF: Auto, Forced Flash, Suppressed Flash, Slow Synchro / Red-eye removal ON: Red-eye Reduction Auto, Red-eye Reduction & Forced Flash, Suppressed Flash, Red-eye Reduction & Slow Synchro.
  • Movie Recording 1280 (1280x720: HD) / 640 (640x480: VGA) / 320 (320x240: QVGA), 30frames/sec. - with monaural sound
  • Shooting functions Instant zoom, Face Detection with red-eye removal, Framing guideline, blink detection, frame No. memory, histogram display, motion panorama
  • Playback functions Face Detection, red-eye removal, multi-frame playback with micro thumbnail, cropping, resize, slide show, image rotation, voice memo, histogram display, exposure warning Photobook Assist, image search, favorites
  • Other functions PictBridge, Exif Print, PRINT Image Matching II, 31 Languages, Time difference, Silent mode, FinePix color, Battery dischage, auto image rotation playback
  • Connectivity - Video output NTSC / PAL selectable; Digital Interface USB 2.0 High-Speed; HDMI Mini connector
  • Power Supply 4xAA type alkaline Batteries (included) / 4xAA type Ni-MH rechargeable batteries (sold separately) / 4xAA type lithium batteries (sold separately)
  • Unit Dimensions 110.2(W) x 73.4(H) x 81.4(D) mm / 4.3(W) x 2.9(H) x 3.2(D) in. (excluding accessories and attachments)
  • Unit Weight Approx. 341g / 12.0oz. (excluding accessories, battery and memory card); Approx. 437g / 15.4oz. (including accessories, batteries and memory card)
