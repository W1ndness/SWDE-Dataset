3.5" touch screen LCD / 4x optical zoom / "Exmor R" sensor / HD movie / iAuto & Scene Recognition / iSweep Panorama Mode / ISO 3200 / Anti-Motion Blur

The sleek and stylish DSC-TX7 offers a large 3.5" touch screen to easily take and review photos. An advanced "Exmor R" CMOS sensor provides stunning low-light images and sweeping panoramic views with iSweep Panorama mode. Plus, enjoy 1920x1080 Full HD Video Recording and playback.
  • Supplied Accessories: Rechargeable Battery (NP-BN), Battery Charger, AV Cable, USB Cable, wrist Strap, Paint Pen, CD-ROM, Multi Output Stand
  • 3.5" Touch screen for easy Focus selection and photo viewing
  • "Exmor R" CMOS sensor for stunning low-light performance
  • iSweep Panorama Mode captures stunning panoramic images
  • Fast capture with 10fps at full 10.2MP resolution
  • 1080i AVCHD Movie records high-quality HD movies
  • Carl Zeiss Vario-Tessar 4x (25mm equivalent) Optical Zoom lens
  • Anti-Motion Blur/Hand-held twilight for less grain and blur
  • Backlight Correction HDR for scenes with high-contrast
  • High Sensitivity ISO 3200
  • Optical SteadyShot® image stabilization
  • Specifications
  • Effective Pixel: Approximate 10.2 MP
  • Imaging Device: 1/2.4 type(7.59mm) Exmor R CMOS Sensor
  • Lens: Carl Zeiss Vario-Tessar (12 elements in 10 groups)
  • Optical Zoom: 4x
  • Focal Length: 4.43-17.7mm (35mm Equivalent: Still shots: 25-100mm / Movie 16:9: 28-112mm / Movie 4:3: 34-136mm)
  • LCD: 3.5" Xtra Fine TruBlack (921K pixels)
  • Recording Media: 45MB internal Flash Memory, Optional Memory Stick® Duo Media, optional Memory Stick® PRO Duo Media, optional Memory Stick® PRO Duo (High Speed), optional Memory Stick® PRO HG-Duo, optional SD/SDHC media
  • Movie Mode(s): Up to AVCHD: 1920 x 1080 (59.94i, Interlace) (Approx.17Mbps) / VGA (640 x 480) (29.97fps, Progressive) (Approx.3Mbps)
  • ISO: Auto / 125 / 200 / 400 / 800 / 1600 / 3200
  • Scene Mode(s): High Sensitivity / Twilight / Twilight Portrait / Soft Snap / Landscape / Beach / Snow / Fireworks / Hi-Speed Shutter / Underwater / Gourmet / Pet
  • Power: InfoLITHIUM N type 3.6V
  • Dimensions (Approx.): 3 7/8" x 2 3/8" x 23/32" (W x H x D)
  • Weight (Approx.): 4.7 oz.
