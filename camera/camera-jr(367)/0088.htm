New slim-line body design / 4x optical zoom / 2.7" color LCD / Image Stabilization / HD Movie / Make-Up Mode / SD/SDHC memory card / Li-ion battery
The Casio EXILIM EX-S200 is the perfect combination of design and technology in an ultra-slim format. The camera, which is as slim as it is elegant, is equipped with amazing functions, such as the new EXILIM engine 5.0, 14.1 Megapixels, 720p HD Video and Premium Auto Mode. Its exclusive Landscape Enhancement Mode includes a mist removal feature to clear up hazy outdoor photos, while Make-Up Mode makes everyone look their best. Available in fun colors, this camera is a perfect reflection of your individual style.

Features
  • New slim-line body design in exciting & fun colors with simplified rear button interface
  • Image Stabilization
  • Premium Auto Mode
  • HD Movie
    Specifications
  • Image Sensor: 1/2.3-inch square pixel CCD / Total Pixels: 14.48 megapixels (/million)
  • Lens: Focal Length: f=4.9 - 19.6mm (35mm Film Equivalent: Approx. 27 - 108mm)
  • Zoom: Optical Zoom: 4x optical zoom, 6x Single SR Zoom
  • Monitor: 2.7-inch TFT color LCD (Super Clear LCD)
  • Recording Media: 34.9MB Built-in flash memory, SD Memory Card, SDHC Memory Card compatible
  • File Format: Still images: JPEG (Exif Ver. 2.3), DCF2.0, DPOF / Movies: AVI format, Motion JPEG, IMA-ADPCM (monaural) / Audio (Voice Recording): WAV format (monaural)
  • Number of Recorded Pixels: Still images: 14M(4320x3240)/3:2(4320x2880)/ 16.9(4320x2432) / VGA(640x480) / Movies: HD (1280X720)/STD (640x480)
  • Built-In Flash: Flash Modes: Auto, Flash off, Flash on, Soft Flash, Red Eye Reduction
  • Recording Functions: Snapshot (Auto mode/ Premium Auto mode), Snapshot by Super Resolution technology, Marco, Self-timer, Dynamic Photo, Continuous shooting (4fps CS/10fps CS/Flash CS), BEST SHOT, Make-up mode, Landscape Effect, Auto shutter, Face Detection, Movie (HD Movie, Normal Movie, Prerecord Movie, and For YouTube), Voice Recording
  • Sensitivity: Still images: Auto/ ISO50/ ISO100/ ISO200/ ISO400/ ISO800/ ISO1600/ ISO3200
  • Input / Output Terminals: USB port (Hi-Speed USB compatible) / AV output terminal (NTSC/PAL)
  • Power Source: Rechargeable Lithium ion Battery (NP-120) x1
  • Dimensions: 3.9 (W) x 2.1 (H) x 0.7 (D) in.
  • Weight: 4.6 oz.
