4x optical zoom / 3.0" Clear Photo LCD, iAuto, Smile Shutter / Face Detection / SteadyShot® image stabilization / In-camera retouching

You don't have to be an expert to take great photos. It's all about making it easier to take beautiful pictures. The DSC-W310 captures amazingly detailed images with a 28mm equivalent wide-angle 4x Zoom lens. And everything is made simpler with a 3.0-inch (diag.) screen and advanced features including Easy mode, iAuto and SteadyShot® image stabilization.
  • In-camera retouching is the quick and easy way to enhance your photos on the go; you can trim, correct red eye and use unsharp masking functions
  • SteadyShot image stabilization compensates for camera shake and helps prevent blur by capturing two photos at high shutter speeds and combining them into a single clear image
  • Face Detection technology detects up to eight individual faces and adjusts focus, exposure, and White balance to help deliver crisp, properly lit images of family and friends
  • Smile Shutter technology captures a smile the moment it happens
  • Optical SteadyShot® image stabilization uses a built-in gyro sensor to detect camera shake and automatically shifts the lens to help prevent blur
  • 3.0" Clear Photo LCD (230,400 pixels)
  • Imaging Device: 1/2.3 type (7.79mm) Super HAD™ CCD
  • Recording Media: 6MB internal Flash Memory; Optional Memory Stick® Duo Media, Memory Stick® PRO Duo Media, Memory Stick® PRO Duo (High Speed), Memory Stick® PRO HG-Duo, SD media
  • Battery Capacity: Approx. 110 minutes / Approx. 220 images
  • Battery Type: Lithium ion / InfoLITHIUM N type 3.6V
  • Flash Effective Range: ISO Auto: Approx.0.5 - Approx.3.0m
  • PictBridge Compatible
  • Scene Mode(s): Intelligent Auto / Easy Shooting / Program Auto / High Sensitivity / Soft Snap / Landscape / Beach / Snow Fireworks / Gourmet / Scenes (Twilight / Twilight Portrait / Backlight / Backlight Portrait / Landscape / Macro / Portrait)
  • White Balance Mode: Auto / Daylight / Cloudy / Fluorescent 1 (White Fluorescent Lighting) / Fluorescent 2 (Natural White Fluorescent Lighting) / Fluorescent 3 (Day White Fluorescent Lighting) / Incandescent / Flash
  • 35mm Equivalent: Still shots: 28-112mm / Movie 4:3: 28-112mm
  • Aperture Range: iAuto (F3.0/F4.3 (W), 2 steps) / Program Auto (F3.0/F4.3 (W), 2 steps)
  • Focal Length: 5-20mm
  • Optical Zoom: 4x
  • Shutter Speed: iAuto (1/4" - 1/2,000) / Program Auto (1" - 1/2,000)
  • Audio/Video Output(s) via Multi-Use Connector
  • HD Output (1080i)
  • USB 2.0 Hi-Speed
  • ISO: Auto / 80 / 100 / 200 / 400 / 800 / 1600 / 3200
  • Supplied Software: Picture Motion Browser Vers 5 + USB Driver (Not supported by Macintosh Operating Systems); PMB Portable Vers 1.1
  • Operating System Compatibility: Windows 2000 Professional SP4, XP Home and Professional SP3, 7, Vista SP1; Mac OS 9.1/9.2/OS X (10.1-10.6)
  • Unit Dimensions: 3-7/8" W x 2-1/4" H x 3/4" D, Weight: 4.3 oz. (body only)
